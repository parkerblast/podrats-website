Game FAQs
When will the Elephalgo Platforming game come out?

The plan currently is to have the game out by the first week in Q2 (first week of April)! Stay tuned on Twitter for sneak peeks and previews of the game!

How will rewards from the game work?

There will be a weekly level and those of the community that complete this level will be entered into a drawing for unique weekly NFT prizes only attainable that week!

How do I play the game?!

The game is currently in beta and can be played through the website using the "Game" tab.

How do I connect my wallet to the game?

To connect your wallet you should first create an account through the website using the "register" page. Once done this will bring up a connect wallet button and then sign in using your MyAlgo wallet and it should connect your wallet!

How can I change my display name?

Your display name can be changed using the user-profile page!

The game isn't running that great, what can I do?

Discord Bot FAQs
How to register?

To register your Elephalgo to your discord account use the /register wallet-address: "your wallet address here". This will then pop up a box that looks like this!

How to opt-in?

Once you have clicked the button "Opt-In Here" you will be taken to the Rand Gallery page of the opt-in asset. Click the "Accept Asset-ID" button. This will then pull up a MyAlgo page where you can finish the transaction.

How to finish registration?

Once you've opted-in on Rand Gallery come back to the discord and wait a few seconds. Next, select "I have opted-in!". If completed successfully you will get a "Successfully verified!" message. That looks like this!

Why should I register my Elephalgo?

By registering you will get the "/flex" command! This allows you to flex your Elephalgos in the #flex-your-elephalgos channel! This command also gives a link to the NFT Explorer and Rand Gallery page!

Why do I need to opt-in?

Doing this allows the bot to verify that you own the wallet, since only the wallets owner can opt-in to assets. This prevents someone from stealing your wallet and using it to get access to discord bot features. So, while it might be a bit more tedious, it is worth it!
